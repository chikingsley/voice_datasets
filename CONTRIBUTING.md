# Contributing

This fork curates open voice/audio datasets and welcomes high-signal additions.

## Add a dataset
Open an issue or pull request with:
* Dataset name
* Direct URL to the dataset landing page/download
* Short description (language, task, size/hours if known)
* Access and license notes (open, gated, academic-only, paid, etc.)
* Suggested quick-index placement (task category and language category)

## Validation rules
Submissions are usually merged when:
* The dataset URL is reachable.
* The link points to the dataset itself (not only a paper or vague mention).
* The dataset is not an obvious duplicate of an existing entry.
* The submission includes enough context to be useful to other builders.
* The full catalog entry is added; quick-index updates are encouraged for high-signal additions.

## Preferred format
Use this bullet format in `README.md`:

`* [Dataset Name](https://example.com) - one-line description.`
